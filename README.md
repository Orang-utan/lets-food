## Let's Food ###

A platform to get dinner with friends (Work In Progress)

![alt text](https://raw.githubusercontent.com/Orang-utan/lets-food/master/docs/demo.gif "Work in Progress")

## Get Started
- Clone this repo
- Cd directory
- touch .env
- yarn install
- yarn dev

## Enviroment File
Here are the variables in the ".env" file:
- ATLAS_URI
- TWILIO_SID
- TWILIO_AUTH_TOKEN
- JWT_SECRET
